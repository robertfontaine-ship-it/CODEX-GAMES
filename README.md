# WRS Quest

**Flagship campaign:** **Corpocalypse: Climb the Corporate Ladder**

WRS Quest is a browser-first narrative arcade RPG that teaches Workplace Readiness Skills through workplace decisions, visible consequences, short job challenges, and promotion-based progression.

## Current milestone

Build the first playable vertical slice: **Floor 1 — Intern Orientation**.

The vertical slice includes:

- comic-style OmniCorp outbreak briefing
- four workplace encounters with consequences
- XP, Trust, and Credits progression
- Inbox Zero timed job challenge
- Elevator Escape promotion gate
- Workplace Readiness Skill tracking
- performance review and promotion result
- replayable alternate outcomes
- browser-local save and settings

## Product principle

This is **not** a quiz game with zombies. Workplace decisions are the gameplay. WRS standards operate as invisible infrastructure behind story consequences, allies, challenge modifiers, progression, and reporting.
